## 简介

### 字节跳动公司介绍

字节跳动致力于成为全球创作与交流平台。我们是全球第一家将人工智能应用到主产品的公司，也是全球发展最快的科技公司之一。成立七年来，我们已经发展成为一个拥有4亿日活用户、十几条产品线，业务范围遍布全球的团队。加入我们，一起和优秀的人做有挑战的事。

### 字节跳动效率工程团队是做什么的

#### 愿景

以字节跳动为试验对象，以管理理念为思想，以创建和选购效率工具及服务为手段，提高公司的运营效率，改善员工工作体验。

#### 团队情况

目前效率工程团队分别在北京，深圳，上海，杭州，武汉，成都等地设有研发中心。团队中有众多优秀、经验丰富的研发工程师，也有来自 Facebook、Amazon、腾讯、阿里巴巴、微软的资深产品经理，还有来自各大公司背景的优秀设计师。希望你也成为这个团队的一员，一起做有挑战的事。

效率工程团队是一个追求效率的团队，不断探索工作方法、管理流程、工具系统等方面的改进，成为提升公司运作效率的核心；平等、有爱的团队氛围，工作上追求极致、务实；团队成员间坦诚沟通，不设边界。

#### 我们做什么产品

[Lark （飞书）](https://www.tmtpost.com/3859694.html?from=timeline&isappinstalled=0)是字节跳动旗下的一款针对to B市场推出的企业协同办公套件产品，包含并不限于 IM 即时通讯、共享日历，及文档在线协作等基础企业协作模块。 该产品所应对的项目规模，业务复杂性和技术复杂性非常之高，同时也是公司未来 to B 市场布局的旗舰产品。我们期望利用互联网的思维和技术优势改变传统企业软件的现状，实现公司内部协同和管理的数据化和智能化。

#### 为什么选择来效率工程团队

目前，团队在极速扩张期，急需各类人才加入，在这里你有机会获得以下发展机遇：

1. 应对复杂的业务和技术挑战，实现技术能力的成长
2. 在业务的快速扩张期获得更多的发展机遇
